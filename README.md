#DevTrackr – Goal and Task Tracking Web Application
A full stack app to track goals, tasks, and learning progress


#Project Overview
DevTrackr is a full-stack web application that allows users to create, track, and manage personal goals and tasks.
The platform includes user authentication, task tracking, and an administrative dashboard that allows administrators to monitor user progress.

The project was built as a full-stack software system using Next.js for the frontend and FastAPI for the backend.


#Features
-User account creation and authentication

-Secure login using JWT authentication

-Create and manage personal goals

-Add tasks under goals

-Administrative dashboard for managing users

-REST API backend

-Database persistence

-Production deployment

#System Architecture

Frontend (Next.js)
        ↓
REST API (FastAPI)
        ↓
Database (SQLite/PostgreSQL)

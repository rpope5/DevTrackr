# DevTrackr

A full stack app to track goals, tasks, and learning progress



\## Tech Stack

\- Frontend: React + TypeScript + Vite

\- API: FastAPI (Python)

\- Worker: Node.js (TypeScript)

\- DB: Postgres

\- Cache/Queue: Redis
